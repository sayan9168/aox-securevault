# AOX SecureVault 🔒

> **Production-Grade Client-Side Encrypted Vault with Interactive 3D UI**

Built under **SayanOX Private Limited** as part of the AOX Web App Series.  
Zero-knowledge architecture. No AI. No backend. Pure cryptographic engineering.

## 🏗️ Architecture

- **Encryption:** AES-GCM + PBKDF2 (Web Crypto API)
- **3D Engine:** React Three Fiber + Drei
- **Framework:** Next.js 14 (App Router)
- **State:** Zustand with encrypted persistence
- **Deployment:** Cloudflare Pages

## ⚡ Features

- Military-grade client-side encryption (zero plaintext storage)
- Interactive 3D vault door with physics-based unlock animation
- Responsive design with automatic 3D degradation for mobile
- Encrypted JSON backup/restore
- Auto-lock on tab blur + idle timeout
- Strict CSP headers
